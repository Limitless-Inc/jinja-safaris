## ⚡ FASTEST START (10 Minutes)

### Step 1: Install (2 min)
```bash
npm install
```

### Step 2: Setup Supabase (5 min)

**Quick Setup:**
1. Go to https://supabase.com → Sign up (FREE)
2. Create new project: "jinja-safaris"
3. In SQL Editor: Copy & paste ALL of `supabase-schema.sql` → Run
4. In Storage: Create 5 buckets (slideshow, activities, gallery, hotels, blog) - make PUBLIC
5. In Authentication: Add user (admin@jinjasafaris.com)
6. In Settings → API: Copy your 3 keys

### Step 3: Configure & Run (3 min)

Create `.env.local`:
```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
SUPABASE_SERVICE_ROLE_KEY=your_key
```

Run:
```bash
npm run dev
```

**✅ DONE! Visit http://localhost:3000**

## 🆘 Need Help?

**Quick Fixes:**

**"npm install fails"**
→ Make sure Node.js 18+ is installed

**"Can't connect to Supabase"**
→ Check .env.local has correct keys

**"Images not showing"**
→ Make Storage buckets PUBLIC in Supabase

**"Can't login to admin"**
→ Create user in Supabase Authentication
